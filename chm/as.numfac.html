as.numfac(dae)R Documentation

Convert a factor to a numeric vector.

Description

Converts a factor to a numeric vector with integers 1, 2, ... for the levels.

Usage

as.numfac(factor)

Arguments

factor The factor to be converted.

Value

A numeric vector.

See Also

factor

Examples

 x <- as.numfac(treats)
